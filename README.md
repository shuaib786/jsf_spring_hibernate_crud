# jsf_spring_hibernate_crud

This is a sample JSF + Spring Boot + Hibernate + Mysql Crud application with authorisation and authentication done with spring security.

# System Requirements

Java 1.8
Spring Boot 2.4.1
Jsf 2.0
Hibernate 4
Primefaces 5
Mysql database
